      i)var can be changed and is function scoped.
      ii)let can be changed 
      iii)const cannot be changed after it is declared.


2️⃣ What is the spread operator (...)?

3️⃣ What is the difference between map(), filter(), and forEach()?
    i)map() changes every item and makes a new array.
    ii)filter() selects some items from the array using a condition.
    iii)forEach() only goes through each item and does not make a new array.
